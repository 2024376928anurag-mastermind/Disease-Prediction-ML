# 🧠 Disease Prediction using Machine Learning

## 📌 Project Overview
This project predicts the possibility of disease using patient medical data. It uses classification algorithms to analyze features such as cell measurements and predicts whether a tumor is malignant or benign.

---

## 📊 Dataset
We use the Breast Cancer dataset from sklearn / UCI repository.

Features include:
- Mean radius
- Mean texture
- Mean perimeter
- Mean area
- Mean smoothness
... and more

---

## 🤖 Algorithms Used
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest Classifier

---

## ⚙️ Steps Performed
1. Data Loading
2. Data Preprocessing
3. Feature Scaling
4. Train-Test Split
5. Model Training
6. Model Evaluation

---

## 📈 Results
All models achieved good accuracy on test data. Random Forest performed best among them.
